### [Nuc](https://github.com/nuclibs/nuc) Sublime Text support

Requires installed [haxe_sublime](https://github.com/AndreiRudenko/haxe_sublime) package

##### Usage
Right click to `project.nuc` file and click `Set as current Nuc project file`
You can change build settings by pressing right click in `project.nuc` file and `Set Nuc build settings`

##### Commands
- Nuc: set project file
- Nuc: set build target
- Nuc: set build settings
- Nuc: build
- Nuc: start server
